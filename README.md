# Mindful Charts

Mindful Charts is a modern, minimalist web application designed to help you cultivate self-awareness and track your emotional and physical well-being over time. By providing a simple yet powerful interface for daily check-ins, the app empowers you to notice patterns, understand your feelings, and build a healthier relationship with your inner world.

The application features a clean, multi-step interface that guides you through logging your physical sensations, identifying your emotions with a detailed three-level emotion wheel, and recognizing your current thought patterns. The data is then visualized on a personal dashboard, offering clear insights into your wellness journey.

## ‚ú® Features

- **Multi-Step Check-in Form:** A guided, three-step process to log sensations, emotions, and thoughts.
  - **Sensation Logging:** Pinpoint physical sensations on a human body map, rate their intensity, and add descriptive notes.
  - **3-Level Emotion Wheel:** An interactive wheel to first select a broad emotional category (e.g., Happy, Sad), then a more specific one (e.g., Joyful, Lonely), and finally choose from a rich vocabulary of nine distinct feelings.
  - **Thought Pattern Identification:** A simple checklist to recognize common cognitive patterns without judgment.
- **Insightful Dashboard:** A personal dashboard to visualize your logged data over time.
  - **Stat Cards:** At-a-glance summaries of your last check-in, most frequent emotion, and weekly sensation intensity average.
  - **Sensation Timeline:** A line chart tracking the intensity of your physical sensations over time.
  - **Emotion Distribution:** A pie chart showing the breakdown of your logged emotions.
  - **Recent Entries:** A list of your most recent check-ins for quick review.
- **Minimalist UI:** A clean, modern, and responsive design built for focus and clarity.
- **Local Storage:** All data is saved directly in your browser's local storage, ensuring your privacy.

## üèÅ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/your_username_/mindful-charts.git
   ```
2. Navigate to the project directory:
   ```sh
   cd mindful-charts
   ```
3. Install NPM packages:
   ```sh
   npm install
   ```

### Running the Application

To start the development server, run:

```sh
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
