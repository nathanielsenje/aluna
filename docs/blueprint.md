# **App Name**: Mindful Charts

## Core Features:

- Sensation Logging: Record physical sensations experienced, including location and intensity.
- Emotion Selection: Choose emotions from a wheel interface.
- Thought Recording: Note the type of thinking patterns observed.
- Wellness Charting: Display logged data over time through interactive charts and graphs.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to promote a sense of tranquility and well-being.
- Background color: Soft, desaturated light gray (#F0F4F8) to provide a clean and non-distracting backdrop.
- Accent color: Muted green (#81C784) for positive reinforcement and key actions.
- Body and headline font: 'Inter', a modern sans-serif, providing excellent readability and a neutral aesthetic.
- Use simple, line-based icons to represent different sensations and emotions, maintaining a minimalist feel.
- Maintain a clean and spacious layout, prioritizing ease of use and data visualization clarity.
- Use gentle transitions and animations for data updates and user interactions, avoiding abrupt changes.
